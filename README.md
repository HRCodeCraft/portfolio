# Harshita Rajput — Portfolio

Personal portfolio website for Harshita Rajput, AI Engineer & Full Stack Developer, built with Next.js 14, Three.js, and Framer Motion.

## Tech Stack

- **Framework:** Next.js 14 (App Router), React 18, TypeScript
- **Styling:** Tailwind CSS, Glassmorphism dark theme
- **Animation:** Framer Motion, GSAP
- **3D / Canvas:** Three.js, React Three Fiber, React Three Drei

## Sections

- **Hero** — Animated typewriter roles, neural network canvas background, floating terminal card
- **Tech Marquee** — Scrolling tech stack showcase
- **About** — Personal timeline (education, experience, key stats)
- **Skills** — Technical proficiencies
- **Experience** — Career journey
- **Projects** — AI/ML project highlights
- **AI Lab** — Research and experimental work
- **Blog** — Articles and writeups
- **Certifications** — Professional certifications
- **Contact** — Contact form

## Getting Started

```bash
# Install dependencies
yarn install

# Run development server
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
yarn build
yarn start
```
